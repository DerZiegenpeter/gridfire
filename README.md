# Grid Fire JFST

**Joint Fire Support Team 2D Simulator**

Prozedurale Karte mit sanften Hügeln & Tälern, UTM-Gitter, Höhenanzeige und Kamera-Steuerung.

## Features
- Prozedurale Heightmap (sanfte Hügel/Täler)
- UTM Grid mit Labels (1km + 200m)
- Live Maus: UTM-Koordinaten + Höhe in Metern
- Kamera: Linksklick ziehen + Mausrad Zoom

## Steuerung
- Linke Maustaste + ziehen = Karte schwenken
- Mausrad = Zoom
- Maus bewegen = Koordinaten + Höhe oben
